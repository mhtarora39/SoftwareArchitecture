Input format in problem statement describes the input format to the particular function, not the actual programme. They might be different. So, input format to the actual programme is:
 
First line contains two integers number of nodes and number of edges.
Next number of edges lines contains three integers, where first one denotes dag_from[i], second one denotes dag_to[i] and third one denotes dag_weight[i].
Next line contains from_node.
Next line contains to_node.
